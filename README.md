# CSD 325 - Advanced Python
Wade Eckert's repository for CSD 325 assignments and code

This repository contains my coursework and programming assignments completed for **CSD 325 - Advanced Python** at Bellevue University. Throughout this course, I expanded upon the Python fundamentals learned in previous coursework by developing larger applications, working with external data sources, implementing object-oriented programming principles, and utilizing Python's extensive standard library.

Assignments progressed from writing structured Python programs to developing applications that process data files, consume web APIs, work with JSON data, implement automated testing, and build graphical user interfaces.
